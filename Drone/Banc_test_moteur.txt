==== Voltage sensor ====
[[https://fr.aliexpress.com/item/Smart-Electronics-NEW-Standard-Voltage-Sensor-Module-Test-Electronic-Bricks-For-Robot-Arduino/32658259839.html?spm=a2g0s.9042311.0.0.uJQ96L|Lien]]

Tension d'entrée: DC0-25 V
Tension mesurée: DC0.02445 V-25 V
Résolution de la tension anaolgique: 0.00489 V
Interface d'entrée: rouge VCC, negative GND

Avec les 3 pin du connecteur, connectez ce module avec votre Arduino, vous pouvez aussi le connecter avec un écran IICLCD1602 LCD pour afficher la tension mesurée.

=== Code de référence ===
{{{code: lang="c" linenumbers="True"
#include <Wire.h>
int val11;
int val2;
void setup()
{
	pinMode(LED1,OUTPUT);
	Serial.begin(9600);
	Serial.println("Emartee.Com");
	Serial.println("Voltage: ");
	Serial.print("V");
}
void loop()
{
	float temp;
	val11=analogRead(1);
	temp=val11/4.092;
	val11=(int)temp;//
	val2=((val11%100)/10);
	Serial.println(val2);
	delay(1000);
}
}}}
